Validate your existing phone number to determine the line type and primary category of use.

What you need to know

  • Requires the U.S. or Canadian phone number that needs to be validated is located in the "Home Phone" and/or "Mobile Number" field.
  • The tag 'Phone: Is Landline' denotes the input "HOME PHONE" number is a landline phone number.
  • The tag 'Phone: Is Mobile' denotes the input "HOME PHONE" number is a mobile phone number.
  • The tag 'Mobile: Is Landline' denotes the input "MOBILE NUMBER" is a landline phone number.
  • The tag 'Phone: Category-Business' denotes the input "HOME PHONE" number is a business landline phone number.
  • The tag 'Phone: Category-Residential' denotes the input "HOME PHONE" number is a residential landline phone number.
  • The tag 'Mobile: Category-Business' denotes the input "MOBILE NUMBER" is a business mobile phone number.
  • The tag 'Mobile: Category-Residential' denotes the input "MOBILE NUMBER" number is a residential mobile phone number.
