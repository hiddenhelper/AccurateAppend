Improve email deliverability and reduce bounces by identifying non-existent, invalid or expired email addresses with advanced verification technology. It works by contacting the email host's server and determining whether the address is active and receiving mail. If an email address is deemed to be undeliverable, we will attempt to add a verified deliverable address.

What you need to know

  • Email verification requires there be an email address present in the "Email - Primary" field.
  • Email append requires there be a first name, last name, U.S. street address, city, state, and zip code.
  • Email addresses that are determined to be unverified or unknown may have a new verified deliverable email address added; in which case the existing email address will be moved to the "Email 2" field, and the new verified deliverable email address will be in the "Email - Primary" field.
  • The tag 'VERIFIEDEMAIL' will be added to each record for email addresses that have been verified as deliverable.
  • The tag 'UNVERIFIEDEMAIL' will be added to each record for email addresses that have been verified as not deliverable.
  • The tag 'UNABLETOVERIFYEMAIL' will be added to each record for email addresses that are not able to be verified.
